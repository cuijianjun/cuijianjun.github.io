继承

一个对象可以使用另一个对象的东西，就叫继承。
一个对象可以使用本不属于自己的东西，就叫继承。
js中的原型就是对继承特性的实现。

继承方式1 - 默认的原型继承( 很常用 )

继承方式2 - 覆写构造函数的显式原型( 很常用 )

继承方式3 - 给显式原型混入属性( 很常用 )

继承方式4 - Object.create( 很少用 )

继承方式5 - 借用Object.create方法覆写显式原型( 很少用 )

继承方式6 - 复合式原型继承( 很少用 )
