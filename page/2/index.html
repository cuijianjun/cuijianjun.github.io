作用域链

变量所有的有效区域(运行环境)，统称为作用域链。

闭包

  • 广义闭包: 可以访问非自身变量的函数。

    • 从这中角度上来说所有的函数都是闭包。
    • 还有一种说法是，使用了非自身变量的函数，那么这就是一个闭包。

  • 狭义闭包: 可以访问非自身局部变量的函数。

    • 从这种角度来说，函数内的函数才是闭包。
    • 还有一种说法，使用了非自身局部变量的函数。
    • 闭包的特点:
      • 利用闭包可以在任何地方操作一个局部变量。

变量的生命周期

一个变量从定义开始，到释放结束，中间存活的过程就叫变量的生命周期

  • 全局变量的生命周期:

    从定义开始，到页面被卸载结束，就是全局变量的生命周期。

  • 局部变量的生命周期:

    一般情况下从定义开始，到函数执行完毕结束，就是局部变量的生命周期。但是可以通过闭包延长局部变量的生命周期。

阻止局部变量的释放

闭包的应用场景

  • 操作私有属性

    • 目的: 为了防止其他地方对该属性进行随意的修改的隐患

  • 编写一段缓存数据的代码

  • 闭包操作保存变量的状态

    • 变量的状态: 指变量不同时期所存储的值

原型链的概念

一个对象继承的所有对象，称之为这个对象的原型对象

原型链的概念：

  • 这是一个描述对象继承结构，比较形象的词汇
  • 一个对象继承的所有对象称之为这个对象的原型链

函数的原型链

fn ==> Function.prototype ==> Object.prototype ==> null

补充：

  • 所有函数.prototype(即默认的显示原型)都继承Object.prototype
  • Object.prototype除外
  • 所有的函数，都继承Function.prototype
  • 对象继承的终点是Object.prototype

面向对象第二天

原型对象

对象的原型对象一定是有终点的

函数默认的原型对象继承结构

实例成员

直接添加在实例身上的就称之为实例成员

  • 构造函数不能直接访问实例成员

静态成员

直接添加到类自己的属性和方法，叫类成员(静态成员);

  • 实例不能直接访问静态成员

优点

  • 使用方便不用创建实例

问题

谁能访问静态成员和实例成员呢？

  • 说白了，就是实例能够访问实例成员，类能够访问类成员
  • 类成员只能由构造函数自己访问
  • 构造函数内添加的实例成员，也只有实例能够访问

面向对象的三大特征

  • js中的封装，就是把一堆相关性的变量和函数组织到一起。
  • js中的继承，就是原型。
  • js中的多态，可以理解是对象的动态变化。

for in

for in 可以遍历出一个对像继承的属性（必须是可枚举的）

静态成员和实例成员

静态成员(类成员)

直接添加给类(构造函数)的属性和方法，称之为静态成员

特点:

  • 不需要创建实例即可通过类调用

实例成员

添加给实例的属性和方法，称之为实例成员。

添加到原型中的属性和方法，本意是让实例使用的所以也可以理解为实例成员。

JQuery分析

在JQuery中

  • $.ajax 为静态成员
  • $(“element”).css();这里面css方法就是实例成员

什么情况下会用到静态成员

  • 如果一些属性和类的关联性比较大，那么可以考虑将其作为静态成员存在
  • 如果一些方法具有通用性，那么可以考虑将其作为静态方法存在。

Object原型上常用的几个方法

  1. hasOwnProperty

    • 作用: 判断一个对象是否(自己)含有某个属性
    • 语法: 对象.hasOwnProperty( 要判断的属性名)

    • 返回值: boolean;

      ​

  2. propertyIsEnumerable

    • 作用: 判断一个对象是否(自己)含有某个属性，并且还要判断这个属性是不是可枚举的
    • 这个方法是一个双重判断，通常这个方法为hasOwnProperty的增强版
    • 语法: 对象.propertyIsEnumerable( 要判断的属性名)

    • 返回值: boolean;

      ​

  3. isPrototypeOf

    • 作用: 判断一个对象是不是另一个对象的原型对象

    • 语法: 对象.isPrototypeOf( 对象)

    • 返回值: boolean;

      ​

  4. toString

    • 作用: 根据方法执行时内部的this指向返回一个类似与这样的字符串:[object this对象的类型名称]
    • 返回值: 一个字符串

    • 通过这个字符串，可以得知内置对象的类型

函数默认的原型对象的类型

很多地方认为构造函数默认的显示原型对象的类型，是构造函数的名字。

即 Person.prototype是Person类型的对象.

但是，RegExp Error 对象的类型在chrome浏览器上类型为Object，在ie上为构造函数名字(不同浏览起不同的实现)

作用域

就是变量的有效范围。

如何检测一个变量的作用

全局变量

在任何地方都可以使用的变量

在函数外定义的都是全局变量

局部变量

只有指定地方可以使用的变量称之为局部变量

在函数内声明的变量都是局部变量

补充

全局变量和局部变量就是通过作用域的大小来对变量进行的种类划分

函数作用域

只有函数能够划分变量的作用域，这种作用域的规则就叫做函数作用域。

补充:

  • 块级作用域: 凡是代码块就可以划分变量的作用域，这种作用域的规则叫做块级作用域 。(在ES6之前js不支持块级作用域)。

词法作用域

补充

块级作用域 函数作用域 词法作用域之间的联系:

  • 块级作用域和函数作用域描述的是，什么东西可以划分变量的作用域，同时块级作用域包含函数作用域
  • 词法作用域描述的是变量的查找规则
  • 关系:
    • 块级作用域包含函数作用域
    • 词法作用域与块级作用域&函数作用域 没有任何关系
      • 他们从两个角度描述的作用域规则，ES6之前采用的是函数作用域+词法作用域
      • ES6采用的是块级作用域 + 词法作用域
